# About Project

This app is a Pomodoro timer featuring three modes: pomodoro, short break, and long break. The mode switches automatically when the timer runs out, though you can also manually switch to whichever mode you prefer. You can pause and resume the timer at any time. Additionally, the remaining time and current mode are displayed in the browser tab while the timer is running. In the settings, you can adjust the duration for each mode, manage sound effects, and change the font and color theme. 

The project was made using React, TypeScript, Redux Toolkit, CSS modules, Webpack and FSD. [See the demo here](https://bemfes.github.io/pomodoro-timer/)

The project also contains unit tests and integration tests written in Jest and React Testing Library.

The project uses commit linting with Сommitlint and Husky. Also, for each commit, linting and code formatting of committed files is performed using lint-staged.
