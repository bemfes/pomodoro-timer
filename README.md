# 🍅 Помодоро Таймер

Веб-приложение для управления временем по технике **Помодоро**. Приложение поддерживает три режима работы (помодоро, короткий перерыв и длинный перерыв), автоматически переключает их после завершения таймера и позволяет полностью настроить продолжительность каждого режима, звуковые уведомления, шрифт и цветовую тему.

🌐 [Демо проекта](https://bemfes.github.io/pomodoro-timer/)

## 🎯 Функционал

* Таймер Помодоро с тремя режимами работы
* Автоматическое переключение между режимами после завершения таймера
* Запуск, остановка и возобновление таймера
* Отображение оставшегося времени и текущего режима во вкладке браузера
* Настройка длительности каждого режима
* Включение и отключение звуковых уведомлений
* Переключение цветовой темы приложения
* Выбор шрифта интерфейса

## 🚀 Установка и запуск

1. Клонируйте репозиторий

```
git clone https://github.com/bemfes/pomodoro-timer.git
```

2. Установите зависимости
```
npm install
```

3. Запустите проект в режиме разработки
```
npm run dev
```
